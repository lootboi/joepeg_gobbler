Functionality:
    1. Create wallets to swarm
    2. Fund wallets to swarm
    3. Mint NFTs
    4. Refund "admin" wallet once NFTs are minted

These commands setup the bot for minting:
Create and fund Wallets:
```shell
npm run swarm
```
Run bot 
```shell
npm run gobble
```
